Google Maps API Integration

Integrate Google Maps directly into your modules by extending the abstract class provided by this module with widget to load the map into a field in the odoo backend.

Main Features:

Supporting module for extension in modules that integrate with Google Maps.

Map Widget in the Odoo backend).

How to Use:

After installation, configure your Google Maps API key in system settings. Next, add widget google_maps on the field It is not possible to see the module's functionality only with it installed
