Introduction

This module is designed to help communities manage and locate recycling points efficiently. It integrates seamlessly with the backend, providing a map view of recycling points, along with their address, description, and accepted waste types. Additionally, it offers a configuration menu for setting up the Google Maps API, enhancing the mapping capabilities.

Features
  • Interactive map displaying recycling points.
  • Detailed information on each point, including address, description, and accepted waste types.
  • Waste type management module for categorizing recyclable materials.
  • Google Maps API integration for accurate mapping and geolocation services.
  • User-friendly portal for finding nearby recycling points based on selected distance filters.
  • Auto-fill map functionality. By clicking on the icon within the input field, the map is automatically populated based on the provided information
Installation and Configuration

Follow these steps to install and configure the Recycling Points Module:

  1. Install the module from the Odoo Apps store.
  2. Access the backend, navigate to the Configuration menu, and set up the Google Maps API key.
  3. Manage waste types and recycling points through the dedicated backend menus.

For detailed setup instructions, refer to the official documentation or contact support.

User Guide

To locate nearby recycling points through the portal:

  1. Access the Recycling Points Portal.
  2. Select the desired distance range to filter the recycling points.
  3. View the results on the map, along with detailed information about each point.
